# Analiza zagrożenia powodziowego i użytkowania terenu w zlewni rzeki Liwiec
Projekt przedstawia analizę przestrzenną zagrożenia powodziowego oraz struktury użytkowania terenu w zlewni cząstkowej rzeki Liwiec położonej w województwie mazowieckim.

Analiza została wykonana z wykorzystaniem oprogramowania QGIS na podstawie ogólnodostępnych danych przestrzennych, w tym danych Corine Land Cover (CLC) oraz map zagrożenia powodziowego.

Wyniki analizy wskazują, że w obszarach zagrożonych powodzią dominują tereny rolnicze, które stanowią 70% powierzchni analizowanego obszaru.

## Wykorzystane narzędzia
* QGIS
* Corine Land Cover (CLC)
* dane hydrograficzne

## Źródła danych

Dane wykorzystane w analizie:
* Corine Land Cover - Europejska Agencja Środowiska
* dane hydrograficzne - BDOT/dane publiczne
* mapy zagrożenia powodziowego - Państwowe Gospodarstwo Wodne Wody Polskie
